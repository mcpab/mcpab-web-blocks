# AI Context

This package is a stable TypeScript architecture-first system.

The primary goal is long-term maintainability and composability,
NOT rapid abstraction churn.

The package favors:

- pure computation layers
- SSR-safe logic
- deterministic transforms
- explicit typing
- architecture separation
- incremental evolution

The package is already functional and stable.

Most future work consists of:
- adding features
- extending policies
- adding transforms
- fixing edge cases
- improving typing
- preserving backward compatibility

Agents should prefer small patches that preserve existing patterns.
