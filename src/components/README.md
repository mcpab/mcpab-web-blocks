# Components

UI-facing blocks and primitives.

## Main groups

- `banner`: hero/banner sections and carousel primitives.
- `buttons`: reusable action buttons.
- `cards`: card-style media/text blocks.
- `content`: hierarchy-driven text drawer components.
- `footers`: preset footer layouts.
- `header`: top navigation/header composition.
- `layout`: structural layout blocks and composed sections.
- `media`: media-focused blocks (e.g., modal video).
- `menus`: drawer and dropdown menu system.
- `navigation`: breadcrumb components.
- `styled`: low-level styled helpers.
- `typography`: title and label components.

Import from `@mcpab/web-blocks` or `src/components`.
